# Vue.js Web App

A simple web app with a Vue.js client that consumes a Node.js + Express API.

### Installation
Clone the repository. In root directory:

```
npm install
```

```
cd client
```

```
npm install
```

To run the application:

in root directory

```
npm run dev
```

```
cd client
```

```
npm run serve
```

View the application on http://localhost:8080/.
